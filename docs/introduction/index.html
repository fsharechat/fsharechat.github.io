项目迁移问题#

NOTE: 针对项目改变地址,可以使用以下方法快速切换代码拉取地址,以下以chat-server-release具体说明,其他项目基本类似

git remote rm origin
git remote add origin git@github.com:fsharechat/chat-server-release.git
## 同步以下主干分支提交
git branch --set-upstream-to=origin/master
